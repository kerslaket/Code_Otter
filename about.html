Hi. Welcome To Code-Otter.

Code-Otter is a platform for teaching anyone how to program, in the best possible way. We do this by incorporating a well written, and well structured course into several different mediums like video, web tutorials, and interactive competitive coding. This course focuses on using the python programming language, as its an incredibly sophisticated language, but its also incredibly easy to set up, and incredibly intuitive to learn. Hopefully one day the course will expand to other languages or computing concepts but for now, whether you're a newcomer whos never touched coding before, or a software engineer looking to touch up their skills there will be something here for you.

Code-Otter was created and designed by Toby Kerslake in 2018 as part of his disortation project while attending the University Of Lincoln in the UK. Tired of courses not being comprehensive enough and not being able to cater to all kinds of learners he decided he wanted to create a course that would allow anyone to learn programming.
