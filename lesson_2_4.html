Generating a Random Number

To Generate a random number were going to use something called a library. A library, simply put is a set of code that you can use so you don’t have to write it yourself. These are usually for some more complicated procedures like generating random numbers, and complex mathmatical functions. For this were using one of pythons inbuilt libraries called “Random”. The syntax for using libraries in python is really easy. Its simply the keyword import followed by the name of your library, in this case random. This must be placed at the top of the python file though, before any code occurs so the interpreter knows where its getting the rest of the code from beforeyou call it. Now we can use any of the functions provided by the random library. This provides many functions but the one were using today, and the one you’ll probably use the most often, is the randint function. This generates a random integer. You can call functions from libraries you've imported by first typing the name of the library, in this case, random, followed by a period, and then the name of the function, followed by 2 brackets. So in this case it should look something this “random.randint()”. The reason for the brackets is because in python all function calls require brackets. These contain the “arguments” for the function. Basically any data that the function relies on to operate. So in this case the randint function requires 2 arguments. A starting number and an end number. The function uses this as a range to generate a number within. So if you put in the number 100 and 102, the only numbers it could generate are 101 and 102. The reason it can’t generate the number 100 is that the function starts at the number after the first argument. So in this case we want our range of numbers to be between 1 and 6, since we are trying to simulate a 6 sided die. So for that we want our arguments to be 0 and 6. So if you had the string “cat “ and the string “in the hat”, concatenating them would result in the one string “cat in the hat”, were going to do a similar thing with our print statement. Start by writing out the start of a print statement to print the users name. So that would look like print(userName).
