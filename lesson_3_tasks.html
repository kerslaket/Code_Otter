Using For Loops

Try creating the programs below. Don’t worry if you struggle the answers are on the next page for reference. Just make sure you don’t cheat, make sure you understand what you’re implementing before moving on to the next section.

Task 1 - Multiplication Identification

Make a program that takes an integer as an input from the user and then write multiple if/elif/else statements. Have the program output a message if the number is a multiple of 4. Have the program output another message if it is a multiple of 5. Have the program output a different message if it is a multiple of both. If the number is a multiple of neither then print yet another message.

Task 2 - Dice Roll Game Extension

Add a for loop to the 2 player dice roll game we made so that each player rolls 3 dice. Then compare the highest score of each player to see who wins.

Task 3 - Text Adventure

A text adventure is a game that relies entirely on outputting text to the user, giving them several options, and then outputting results based on the decision the user made. Similar to a choose your own adventure book. These were commonly made games back before computers were capable of intensive graphics and relied heavily on the users imagination. Do some research into these, particularly looking up the game Zork, and make your own version using If Else statements. No need to go overboard feel free to limit yourself to just 2 or 3 paths.

Extension

Add a while loop to your text adventure so that when the user dies or fails they get the option to go back to the beginning and start again.

Solutions

Task 1 - Multiplication Identification

num = int(input("Please Enter A Number: ")) if (num % 4 == 0) and (num % 5 == 0): print(str(num) + " is a multiple of 4 and 5") elif (num % 4 == 0): print(str(num) + " is a multiple of 4") elif (num % 5 == 0): print(str(num) + " is a multiple of 5") else: print(str(num) + " is not a multiple of 5 nor 4")
