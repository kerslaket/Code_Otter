Conditionals? What Are They?



Conditionals in programming are essentially a way of making decisions in your program and controlling the programs flow. There is 3 main components to this, If Statements, For Loops, and While Loops. If Statements allow you to execute specific blocks of code only if a certain condition is met and then execute other specific blocks if they aren’t. This can be used to accomplish many things, for example printing a certain result based on an input. For loops allow you to continuously repeat a block of code for a predetermined amount of time. This allows you to write less code and have a more efficient program since you don’t have to repeat lines of code that you may need to reuse. While Loops accomplish a similar thing except the code is reiterated continuously until a certain condition is met.
