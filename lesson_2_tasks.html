First Exercises

Try creating the programs below. Don’t worry if you struggle the answers are on the next page for reference. Just make sure you don’t cheat, make sure you understand what you’re implementing before moving on to the next section.

Task 1 - Complimenter Program

For this simple program, just have a user enter their name, and reply to them with a compliment. For example if the user enters the name “Toby” your program could reply “You Look Very Handsome Today Toby”.

Hint: You should probably only require one variable and a print statement. The variable should be used to hold a string given by the user.

Task 2 - Age Calculator

Create a program that has a user enter their age in years and then outputs their rough age in days. Try to make the user experience as pleasant as possible.

Hint: You’ll need to store an integer input by the user and then times it by 365. Or if you wanna be a pretentious ass 365.25.

Task 3 - Multiplier

Create a program that takes two integers from the user and multiplies them together, and then returns the output.

Hint: Nice try you’re on your own for this one

Extension

Improve the multiplier program by also returning the result of the two numbers being divided, maybe even add some other mathematical functions just for fun. Then maybe allow for the use of more than just two numbers.

Solutions

Task 1 - Complimenter

name = input("Please Enter Your Name: ") print("Hi " + name + ", You look really handsome today")

Task 2 - Age Calculator

age = int(input("Please Enter Your Age")) print("You are " + str(age * 365.25) + " Days Old")

Task 3 - Multiplier

num1 = int(input("Please Enter a Number: ")) num2 = int(input("Please Enter another Number: ")) print(str(num1) + " multiplied by " + str(num2) + " is " + str(num1 * num2))

Extension

num1 = int(input("Please Enter a Number: ")) num2 = int(input("Please Enter Another Number: ")) num3 = int(input("Please Enter yet Another Number: ")) print(str(num1) + " multiplied by " + str(num2) + " multiplied by " + str(num3) + " is " + str(num1 * num2 * num3)) print(str(num1) + " divided by " + str(num2) + " divided " + str(num3) + " is " + str(num1 / num2 / num3))
